# Phone Number Slider  

## Overview  
This project provides a stylish, interactive slider to select a 10-digit phone number between **1,000,000,000** and **9,999,999,999**. Built with **HTML, CSS, and JavaScript**, it features a clean, modern design with smooth animations and responsive layout.  

## Features  
- **Range Slider**: Smoothly select any 10-digit number within the specified range.  
- **Formatted Display**: Shows the number in both **comma-separated** (e.g., `1,234,567,890`) and **phone number format** (e.g., `+1 (234) 567-8900`).  
- **Copy Functionality**: One-click copy with a confirmation tooltip.  
- **Responsive Design**: Works on desktop and mobile devices.  
- **Modern UI**: Clean, animated interface with a blue-themed color scheme.  

## How to Use  
1. **Slide** the range input to select a number.  
2. View the formatted number in real time.  
3. Click **"Copy Number"** to copy the phone number to your clipboard.  

## Customization  
Modify the CSS variables (`--primary-color`, `--secondary-color`, etc.) to change the theme. Adjust the slider thumb size, shadows, and animations for different effects.  

## Compatibility  
Works on all modern browsers (Chrome, Firefox, Safari, Edge).
